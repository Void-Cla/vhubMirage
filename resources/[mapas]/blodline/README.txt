Installation Instructions:
For Single Player
A - Extract the files and then copy the files from the steam folder and set it on your desktop
B - Open OpenIV, install it if you dont have it already
C - [Make sure you go into edit mode!!]
D - Click update> x64> dlcpacks> custom_maps>dlc.rpf> x64> levels> gta5> citye> maps> custom_maps.rpf
E - Then take the files and drag & drop it into OpenIV then its in!

FiveM:
Place the entire resource folder in your server’s resources directory.
Add ensure my-map-resource to your server.cfg.
Restart the server.
